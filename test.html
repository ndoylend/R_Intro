Now Let's Look at Text Size

Normal

What's the default size, and where in css is it defined? Well, this is 40px and this is 26px. 26px is the default then. The default line height of 1.5 em keeps the text looking nice and readable.

Bigger

If the default font size is 26px, then the bigger size should be something like 32px. Keeping the default line height of 1.5 em keeps the text looking nice and readable.

Smaller

If the default font size is 26px, then the bigger size should be something like 22px. Keeping the default line height of 1.5 em keeps the text looking nice and readable.
